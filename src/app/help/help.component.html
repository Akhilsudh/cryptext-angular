To Encrypt Content
  • Navigate to the Encrypt section on the Navigation Bar at the top
  • Enter the text that needs encrypting in the Rich text editor field.
  • Enter a password and confirm the same in the confirm password field.
  • Enter a file name for the encrypted text file.
  • On clicking the Encrypt button, a file with the name provided will be generated. This file can be provided for decryption along with the password mentioned earlier
    •      To Decrypt Content
  • Navigate to the Decrypt section on the Navigation Bar at the top.
  • Drag an encrypted file into the drop zone or click on the zone to select a file to decrypt.
  • Enter the password in the password field.
  • Click on decrypt content button.
  • The decrypted content can be seen in the rich text field which can be edited and re-encrypted with a new password value and file name.
    •      About CRYPText is a tool that helps in encrypting content and generate a text file that can be decrypted with a previously specified password. CRYPText uses AES(Advanced Encryption Standard). It is a symmetric key algorithm, which means that the same password used to encrypt a file is used to decrypt it.
