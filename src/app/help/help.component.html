Select one fo the following to

  • Navigate to the Encrypt section on the Navigation Bar at the top
  • Enter the text that needs encrypting in the Rick text editor field.
  • Enter the password and confirm the same in the confirm password field.
  • Enter a file name for the encrypted text file.
  • On clicking the Encrypt section, a file with the name provided will be generated. This file can be provided to the decrypt section
  • Navigate to the Decrypt section on the Navigation Bar at the top.
  • Drag an encrypted file into the drop zone or click on the zone to select a file to decrypt.
  • Enter the password in the password field.
  • Click on decrypt content button.
  • The decrypted content can be edited and re-encrypted with a new password value and file name.
